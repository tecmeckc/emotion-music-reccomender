🎧 Emotion-Aware Music Recommendation System

An AI-powered application that detects user emotions and recommends music accordingly, providing a personalized and engaging user experience.

🚀 Features
🎭 Emotion detection from user input
🎵 Music recommendations based on detected emotions
🤖 Machine Learning-based prediction
🌐 Simple and interactive UI
⚡ Fast and real-time results

🧠 How It Works
User provides input (text / selection / emotion input)
The system processes the input using a trained ML model
Emotion is classified (e.g., Happy, Sad, Angry, Neutral)
Music is recommended based on the detected emotion
Results are displayed to the user

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python / Node.js
Machine Learning: Python (Scikit-learn / NLP)
Libraries: Pandas, NumPy

⚙️ Installation & Setup
Clone the repository
git clone https://github.com/tecmeckc/emotion-music-reccomender.git
Navigate to the project folder
cd emotion-music-reccomender
Install dependencies

For Python:
pip install -r requirements.txt

For Node.js:
npm install

Run the application
For Python:
python app.py

📊 Example
Emotion → Recommendation
Happy → Energetic / Pop
Sad → Calm / Acoustic
Angry → Rock / Intense
Relaxed → Instrumental
