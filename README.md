# 🎧 Emotion-Aware Music Recommendation System

An AI-powered application that detects user emotions and recommends music accordingly, providing a personalized and engaging user experience.

---

## 🚀 Features

- 🎭 Emotion detection from user input  
- 🎵 Music recommendations based on detected emotions  
- 🤖 Machine Learning-based prediction  
- 🌐 Simple and interactive UI  
- ⚡ Fast and real-time results  

---

## 🧠 How It Works

1. User provides input (text / selection / emotion input)  
2. The system processes the input using a trained ML model  
3. Emotion is classified (e.g., Happy, Sad, Angry, Neutral)  
4. Music is recommended based on the detected emotion  
5. Results are displayed to the user  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python / Node.js  
- **Machine Learning:** Python (Scikit-learn / NLP)  
- **Libraries:** Pandas, NumPy

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/tecmeckc/emotion-music-reccomender.git
cd emotion-music-reccomender
```

### 2. Install dependencies

**For Python:**

```bash
pip install -r requirements.txt
```

**For Node.js:**

```bash
npm install
```

### 3. Run the application

**For Python:**

```bash
python app.py
```
